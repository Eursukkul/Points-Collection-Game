# เกมสะสมคะแนน Nextzy — Points Collection Game

Gamification web app — ผู้เล่นกดสุ่มคะแนน (300 / 500 / 1000 / 3000) สะสมจนถึงเพดาน 10,000 คะแนน แล้วปลดล็อกรางวัลตาม checkpoint (5,000 / 7,500 / 10,000) พร้อมดูประวัติการเล่นและประวัติรางวัล

## Tech Stack

| Layer | Stack |
|---|---|
| Frontend | Next.js (App Router) + Tailwind CSS — `frontend/` |
| Backend | Go + fiber + Gorm — `backend/` |
| Database | PostgreSQL |
